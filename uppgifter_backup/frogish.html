1. När startade utvecklingen av Internet? Vad startade utvecklingen?

World Wide Web uppfanns 1991 av britten Tim Berners-Lee. Vilket gjorde det möjligt att nu skapa egna hemsidor och surfra runt bland andra och deras nät.

2. Vad var Internet tänkt för i början?

Internet handlade från början om att få datornätverk att kommunicera. Skicka meddelande från dator till dator, likt hur dagens email fungerar.

3. Vad är ARPA?

ARPA Net, The Adavanced Research Projects Agency Network, var den första wide-area packet-siwtched network med distribuerad control och en av de första nätverk som implementerade TCP/IP protkoll.

4. Vad heter nätet som var grunden till Internet?

Internets rötter komm från ARAPANET som skapades för att bättre kunna uttnyttja landets datorkapacitet till försvarsforkning under kalla kriget. Fler datornätverk bygges ihop i något som kallades "The internetting project"

5. Varför kunde ingen dator vara centraldator?

Den främsta anledningen till varför man inte hade kvar en centraldator var pågrund av säkerhetsskäl. Om någon inkräktare skulle sabotera centraldatorn eller om det skulle uppstå naturliga problem med centraldatorn så skulle hela nätverket ligga nere.

6. Vad är fördelen med distribuerat nätverk i stället för centraliserat?

Fördelen med ett distribureat nätverk är att man inte behöver ha någon källa eller ett center, dator kommunikationen sker istället direkt mellan avsändare och mål. Medans i ett centraliserat system så måste man vara uppkopplad till en källa som alla kommunikation går igenom, vilket är väldigt ineffektivt.

7. Vad står www för?

www står för World Wide Web eller webben som det också kallas är det ställe där alla dagens hemsidor är uppladdade, på en server som sedan är sammankopplade med andra server som tillsammans utgör internet.

8. Vad är IP-adress och hur fungerar de?

IP står för internet protokoll och ger en unik identitet till varje elektronisk enhet som är uppkopplad till internet. En ip-adress används tillsammans med olika typer av protokoll för att överföra data på det angivna sättet.

9. Vad är syftet med TCP?

Syftet med TCP är att skicka information i små packet som sedan packas upp och sammanställs vid målenheten. Motsatsen till TCP är UDP vilket är ett snabbare sätt att överföra information, men det är ett mindre pålitligt sätt eftersom säkerheten är sämre och delar av infon kan gå tappad vid överföring

10. När blev Internet populärt för vanliga mäniskor?

Internet spreds sig särskilt snabbt i Sverige om man jämför med andra länder. 1993 använde ungerfär 2% av befolkiningen internet och bara 5 år senare låg siffran på 33%, vid millenieskiftet så hade den siffran ytterliggare ökat till hela >50%

11. Med vilket protokoll kommunicera man över internet?

TCP/IP protokollet är det som används på internet idag, för att datorer ska kunna kommunicera med varandra.

12. Vilket språk var föregångaren till HTML?

SGML (Standard Generalized Markup Language) var det som användes innan HTML

13. Vad var Gopher?

Gopher är en typ av internetprotokoll som innehar möjligtheten att extrahera och öppna web document som är lagrade på virtuella webservrar. Gopher är en av internets första data och fil återkomst protocol som användes för att köras ovanpå ett TCP/IP nätverk.

14. Vilka var de första webbläsarna?

Den absolut första populära webbläsaren var "Mosaic" och lanserades år 1993. Kort därefter så kom Netscape Navigator och sedan Internet Explorer, som än används idag (även om den inte är speciellt bra).

15. När började vanliga personer att använda datorer?

Under 1990-talet började privatpersoner använda nätet i större utsträckning, då kommersiella operatörer erbjöd förbindelser åt hemanvändare och World Wide Web slog igenom.

16. Vad används internet till idag?

Idag så används internet till en rad olika saker. Däribland främst kommunikation till människor runt om i världen, fakta framtagning och underhållning

17. Vad är en URL?

URL står för Uniform Resource Locator och är en sorts filsökväg till filer eller webadresser.

18. När kom HTML resp. CSS ?

Den första versionen av HTML kom ut 1993. Men det tog lång tid innan första css versionen släpptes, hela 5 år senare.

19. Vad är syftet med CSS?

Syftet med CSS är att kunna stilge websidor för att de ska vara lättare för användaren att använda och kunna ge varje webbsida ett eget unikt utseende.

20. Vad är XML. Och vad är skillnaden mellan XML och HTML.

Xml är ett kodspråk och ett filformat som är skapat för att lagra och överföra data.
HTML- och XML-dokument innehåller data som omges av taggar, men där slutar likheten mellan de två språken. I HTML definierar taggarna utseendet på data-rubrikerna, var stycket börjar osv, medans i XML definierar taggarna strukturen och innebörden av datan.

21. Vem ser till att internet börjar fungera om det "går sönder"?

Det finns ingen enda person eller organisation som ansvarar för hela internet. Eftersom internet är uppbyggt att tusentals olika servrar som varje drivs på olika sätt och av olika personer/organisationer.

22. Vem uppfann Java? Vad hade JAVA för betydelse för Internet?

Java uppfanns av James Gosling på Sun Microsystems under 90-talet. Ursprungligen så döptes språket till D för att sedan ändrar till Oak, men bytte sedan ytterliggare en gång till Java och presenterades därefter till världen år 1995.

Java hade en stor betydelse för internet eftesom språket erbjöd stora biblotek med färdiga datastrukturer och ramverk, bland annat för nätverksanslutning och för att ivsa grafiska fönster på enkelt sätt.
Java kom att bli ett av världens mest populära programmeringsspråk. Plattformsoberoende, objekt-orienterat och enkelt att använda. Kan användas för att programmera allt från appar till robotar.

23. Vad var en Applet (i Java sammanhang)?

I java så är en applet, vilken liten applications som helst som utför en specifik uppgift som körs inom en dedikerad widget motor eller ett större program.

24. Vad är ett serverspråk? Vilka finns?

Ett serverspråk är de språk som hanterar program eller data via en server. Den behöver inte klient för att köras och hanterar oftast digitala bekräftelser eller synca ihop olika klienter.
De mest populära server-sidiga programmeringsspråken är: PHP, Python, Ruby, C# och Javascript (NodeJS).

26. Vad betyder ASP resp. JSP ?

ASP står för "Application Service Provider" och betyder programmvaruuthyrning, det vill säga en tjänst som används via interne och som lagrar data som sparats i tjänsten på distans.
JSP är en java-tenkik som används för att dynamiskt skapa svar från en webbserver, det vill säga en metod för att strukturera datorprogram.

27. Vad är Javascript? Vad är skillnaden på Javascript och serverspråken?

Javascript kan köras som både klient och serverspråk. JAvascript används främst i webben för att skicka data mellan klient och server. Men javascript kan även stå för att visa/gömma element i hemsidor och göra avancerade animationer.

34. Vad är Creative commons?

Symbol 1 Erkännande
Licenstagare får kopiera, distribuera och visa verket samt göra bearbetningar och remixer av det endast om upphovsmannen eller licensgivaren ges erkännande på det sätt som anges av klausulen.

Symbol 2 Dela lika
Licenstagare får endast distribuera bearbetningar av verket under en licens som inte är mer restriktiv än den licens under vilken originalverket publicerades.

Symbol 3 Icke-kommersiell
Licenstagare får kopiera, distribuera och visa verket samt göra bearbetningar och remixer av det endast för icke-kommersiella ändamål.

Symbol 4 Inga Bearbetningar
Licenstagare får kopiera, distribuera och visa endast ordinarie kopior av verket, inga bearbetningar eller modifikationer av det.

35. Vad innebär GDPR?

Dataskyddsförordningen, GDPR, stärker rättigheterna som privatperson. Man har bland annat rätt att få reda på vilka personuppgifter som ett företag eller annan organisation hanterar, rätt att få felaktiga personuppgifter korrigerade och i vissa fall rätt att få uppgifter raderade.
GDPR är ett försök att reglera hur data som gäller enskilda individer får behandlas inom EU. Förordningen syftar till att införa ett nytt tänkande där en central idé är att säkerställa den enskilda individens möjlighet till kontroll över sina egna personuppgifter.
